# Kash

| Learning Project

Kash is a cutting-edge e-wallet app designed for seamless and secure financial transactions. With Kash, you can quickly
and easily make payments, transfer funds, and manage your budget all from your smartphone. Key features include
top-notch security, integration with a wide range of merchants, and convenient expense tracking. Experience effortless
transactions and complete control over your finances with Kash!

## Run Locally

Clone the project

```bash
  git clone https://github.com/adikurniawanid/springboot-kash-api.git
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`spring.application.name`

`spring.datasource.driver-class-name`
`spring.datasource.username`
`spring.datasource.password`
`spring.datasource.url`
`spring.datasource.type`
`spring.datasource.hikari.minimum-idle`
`spring.datasource.hikari.maximum-pool-size`

`spring.jpa.properties.hibernate.format_sql`
`spring.jpa.properties.hibernate.show_sql`

`bcrypt.pepper`

`spring.mail.host`
`spring.mail.port`
`spring.mail.username`
`spring.mail.password`
`spring.mail.properties.mail.smtp.auth`
`spring.mail.properties.mail.smtp.starttls.enable`

`api.security.token.secret`

`springdoc.swagger-ui.path`

`spring.devtools.restart.enabled`
